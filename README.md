## How to deploy (free, takes 2 minutes)

**Option 1 — Netlify (recommended):**
1. Go to [netlify.com](https://netlify.com) and sign up
2. Drag and drop your project folder onto the dashboard
3. Done — you get a free `.netlify.app` URL instantly

**Option 2 — GitHub Pages:**
1. Push this repo to GitHub
2. Go to Settings → Pages
3. Set source to `main` branch → Save
4. Your site is live at `https://yourusername.github.io/portfolio`
